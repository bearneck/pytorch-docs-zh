# PyTorch 治理 \| 维护者

## 职责

- 对分配给模块或库的高优先级问题进行分类和修复
- 对分配给模块或库的高优先级拉取请求进行分类、审查和合并
- 在 [discuss.pytorch.org](https://discuss.pytorch.org/) 和 [dev-discuss.pytorch.org](https://dev-discuss.pytorch.org/) 上回答模块或库相关问题
- 维护公共用户文档和开发文档
- 主持会议，并每半年或每季度分享会议纪要和路线图
