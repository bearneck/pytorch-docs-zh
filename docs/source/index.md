PyTorch 文档
===================================

PyTorch 是一个针对使用 GPU 和 CPU 进行深度学习的优化张量库。

本文档中描述的功能按发布状态分类：

**稳定版 (API-Stable):**
这些功能将得到长期维护，通常不应存在重大的性能限制或文档缺失。我们也期望保持向后兼容性（尽管可能会发生破坏性更改，并会提前一个版本发布通知）。

**不稳定版 (API-Unstable):**
包含所有正在积极开发中的功能，其 API 可能会根据用户反馈、必要的性能改进或操作符覆盖范围尚未完成而发生变化。这些功能的 API 和性能特征可能会改变。
