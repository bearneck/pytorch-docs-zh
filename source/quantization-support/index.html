    torch.ao.quantization¶

    顶层 API¶

    API 说明
    torch.ao.quantization.utils.quantize Quantize the input float model with post training static quantization.
    torch.ao.quantization.utils.quantize_dynamic Converts a float model to dynamic (i.e.
    torch.ao.quantization.utils.quantize_qat Do quantization aware training and output a quantized model
    torch.ao.quantization.utils.prepare Prepares a copy of the model for quantization calibration or quantization-aware training.
    torch.ao.quantization.utils.prepare_qat Prepares a copy of the model for quantization calibration or quantization-aware training and converts it to quantized version.
    torch.ao.quantization.utils.convert Converts submodules in input module to a different module according to mapping by calling from_float method on the target module class.

    torch.ao.quantization.utils.check_min_max_valid Checks if the given minimum and maximum values are valid, meaning that they exist and the min value is less than the max value.
